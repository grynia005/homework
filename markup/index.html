Селектори CSS

Селектор - це частина CSS-правила, яка повідомляє браузеру, до якого елементу веб-сторінки буде застосований CSS стиль.

  • Селектор універсальний "*"

    Універсальний селектор. Обирає всі елементи на сторінці.

  • Селектор елемента (теговий)

    Теговий селектор дозволяє ставити стилі для всіх елементів даного типу (тега).
    Наприклад: "p{}"

  • Селектор класа (class)

    Селектор класу дозволяє задавати стилі для одного і більше елементів з однаковим ім'ям класу, розміщених у різних місцях сторінки. !!! Ви можете використовувати будь-яке ім'я для вашого класу, але воно не повинно починатися з цифри.

  • Селектор ідентифікатора (id)

    Селектор ідентифікатора дозволяє задавати стилі для конкретного елемента (id повинен бути унікальним!). Записується він майже так само, як і клас, тільки в CSS замість крапки ставиться символ решітки #:

  • Селектор груп

    Використовується якщо потрібно задати стилі для декількох елементів веб-сторінки.
    Синтаксис: a, b, c {...}

  • Селектор нащадка

    Селектор нащадка - дозволяє задавати стилі до елементів, які розташовані всередині елемента-контейнера.
    Синтаксис: a b { ... }

  • Дочірній селектор

    Дочерний селектор - позволяет задавать стили к элементам, которые являются прямыми потомкам (т.е. идут сразу за родительским элементом и между ними нет других элементов).
    Синтаксис: a > b { … }

  • Сусідні селектори

    Сусідні селектори – вибирає елемент, який знаходиться безпосередньо після зазначеного елемента, якщо у них спільний батько. Синтаксис: a + b { … }

  • Споріднені селектори

    Споріднені селектори - вибирає всі елементи, які знаходяться після зазначеного елемента, якщо у них спільний батько. Синтаксис: a ~ b { … }

  • Cелектор атрибута

    Cелектор атрибута – дозволяє задавати стилі до елементів на основі імені атрибута або значення атрибута.
    Синтаксис: a[attr=”value”] { … }
